# Case 2 — Olist E-Commerce Analysis

## Business Case
Olist is a Brazilian marketplace connecting sellers with customers. Analysis of 99K+ transactions to identify revenue opportunities, customer behavior, and logistics efficiency.

## Tools
SQL Server · Power BI

## Dataset
- Source: Brazilian E-Commerce Public Dataset (Kaggle)
- 99,441 orders · 112,650 items · 99,441 customers

## Key Findings
- GMV of R$1.54B across 2016–2018
- 97% of customers did not repurchase (retention opportunity)
- 21-day logistics gap between best and worst performing states
- São Paulo generates 5x more revenue than any other state
